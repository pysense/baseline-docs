## 本地预览文档

1）【可选】安装 docsify

如果不需要创建及在本地预览生成的文档，则无需安装 docsify。

```
npm i docsify-cli -g
```

2）启动 HTTP 服务

在本地启动 HTTP 服务实时预览，可使用任何 HTTP 服务或通过 docsify 启动 HTTP 服务（默认访问地址 http://localhost:3000）。

```
cd baseline-docs
docsify serve docs
```
