## 本地预览文档

1）安装 docsify

如果不需要创建及在本地预览生成的文档，则无需安装 docsify。

```
npm i docsify-cli -g
```

2）启动 HTTP 服务

通过 docsify 启动服务，默认访问地址 http://localhost:3000，也可以通过其他程序启动 HTTP 服务，方便实时预览效果。

```
cd baseline-docs
docsify serve docs
```
